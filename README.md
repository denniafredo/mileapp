## Setting on .env
- MONGO_DB_HOST=127.0.0.1  
- MONGO_DB_PORT=27017  
- MONGO_DB_DATABASE=mileappdb  
- MONGO_DB_USERNAME=  
- MONGO_DB_PASSWORD=  
- L5_SWAGGER_CONST_HOST=http://project.test/api/v1

## Next step run this command:
- composer install
- php artisan db:seed
- php artisan serve
